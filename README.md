# Kruti Jasani Portfolio

A React and TypeScript portfolio for Kruti Jasani, a Full Stack Developer focused on production-ready MERN and Next.js applications, scalable REST APIs, and real-time systems.

## Scripts

- `npm run dev` starts the local Vite dev server.
- `npm run lint` runs ESLint.
- `npm run build` creates the production build.

## Resume Data

The portfolio content is based on Kruti's resume, including contact details, education, Oasis Infobyte internship experience, projects, technical skills, and achievements.

## Contact Form Storage

The contact form posts to `/api/contact` and stores submissions in MongoDB Atlas through a Vercel serverless function.

Set these environment variables before deploying:

- `MONGODB_URI`
- `MONGODB_DB_NAME`
- `MONGODB_COLLECTION`
